## ChatGPT and Prompt Basics

---

**ChatGPT** is one of the most widely used AI systems, powered by OpenAI's GPT models. GPT stands for **Generative Pre-trained Transformer:**

  - **Generative** > it generates data
  - **Pre-trained** > trained on large amounts of data
  - **Transformer** > the underlying model architecture

ChatGPT supports **multiple modalities**, meaning inputs aren't limited to text. You can combine text with images, and recent versions also support voice input that feels increasingly like natural conversation.

A key part of what makes ChatGPT effective at following instructions is **RLHF (Reinforcement Learning from Human Feedback)**, a training process where human evaluators rate model responses, helping the model improve over time.

---

**Useful prompting techniques:**

**Assigning a persona**: 

Starting a prompt by giving ChatGPT a role (e.g., "You are a tech marketing expert") helps focus its responses toward a specific context or tone.

**Image generation**: 

ChatGPT has DALL-E integrated, so you can request image generation directly. For better results, include specific details about style, colors, and layout in your prompt.

**Delimiters**: 

When a prompt includes a large block of text or code, use delimiters to clearly separate sections and signal to the model where an example begins and ends. Common options:
  - Triple quotes `"""..."""` for blocks of text
    
  - Markdown style backticks for code (e.g., ` ```python ... ``` `)

Delimiters make complex prompts easier for the model to parse and generally lead to better, more accurate outputs.
